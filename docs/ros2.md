# ROS 2

!!! abstract

    The Robot Operating System (**ROS**) 是一组用于构建机器人应用程序的软件库和工具。从驱动程序和最先进的算法到强大的开发人员工具，ROS 拥有您下一个机器人项目所需的开源工具。

    自 2007 年 ROS 启动以来，机器人和 ROS 社区发生了很多变化。ROS 2 项目的目标是适应这些变化，利用 ROS 1 的优点并改进不足之处。
